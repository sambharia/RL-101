# How LLMs Actually Work — Blog Series

A from-scratch guide to understanding LLMs, written while learning. Each post teaches ONE idea well enough that you could explain it over coffee.

**Audience**: People who can code, know basic calculus, use ChatGPT daily — but don't know what's inside.

**Style**: Running example (names dataset from microGPT), real math where it matters, learning in public. Tutorial, not survey.

**Inspiration**: Karpathy (show the thing working), Alammar (one running example), swyx (opinionated, not precious), 0xkato (clean structure, tiny explainers).
