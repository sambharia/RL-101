# microGPT added as Phase 0 capstone

The learner wants to be able to write microgpt.py (Karpathy's 199-line pure-Python GPT) entirely
from scratch by the end of Phase 0. This is a concrete, achievable milestone and maps perfectly
to their "teach it back" mission.

Curriculum updated:
- Lesson 0002 → covers lines 185–199: the inference/generation loop
- Lesson 0003  → covers lines 29–71 (autograd) + 150–183 (training loop, Adam)
- Lesson 0003b → covers lines 73–143: gpt() forward pass (embeddings, attention, MLP)
- Lesson 0005b → CAPSTONE: write all 199 lines from memory

Decision: NOT added to Lesson 2 (would overload working memory). The file is referenced
at the bottom of Lesson 2 as "this is what you're building toward" but not taught inline.
